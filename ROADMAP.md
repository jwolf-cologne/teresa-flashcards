# Teresa Flashcards Roadmap

This roadmap tracks likely next steps after the first App Store release.

## Version 1.0

Goal: ship a stable first App Store version.

- Manual flashcard learning.
- Deck overview and deck detail pages.
- Deck-level statistics.
- Question list per deck.
- Card creation and deck creation.
- Deck language selection.
- Audio playback for questions and answers.
- Optional AI subscription.
- AI card generation.
- AI study helpers.
- iCloud sync when available.
- Public support, FAQ, and privacy pages.

## Version 1.0.x

Goal: fix issues discovered during App Review or immediately after release.

- App Store review feedback.
- Paywall and StoreKit edge cases.
- Small UI text corrections.
- Crash fixes.
- Minor accessibility improvements.
- Small website/support corrections.

## Version 1.1 Ideas

Goal: improve first-time use and daily learning.

- Better onboarding for new users.
- More polished empty states.
- Optional sample decks per subject.
- More detailed learning statistics.
- Improved review scheduling options.
- Better language handling for mixed-language cards.
- More speech voice options.
- Export or backup option for decks.
- More website languages if needed.

## Later Ideas

Larger ideas that should wait for real user feedback.

- Import cards from CSV, text, or clipboard.
- Share decks with other users.
- Public or private deck templates.
- Tags in addition to decks.
- More advanced spaced repetition algorithm.
- Widgets or reminders.
- iPad-optimized layout.
- macOS version.
- Collaboration or family sharing workflows.
